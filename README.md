<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
The project consists of a simple PowerShell script that builds a functional Active Directory home lab using VirtualBox. This lab demonstrates setting up a domain controller, configuring network adapters, and installing essential services like DHCP. A PowerShell script automatically creates numerous user accounts.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Oracle Virtual Box</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
